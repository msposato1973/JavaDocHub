Pattern Matching for switch

Java 17 → Switch expression without patterns
Java 21 → Pattern matching on switch (stable)
Java 17
// Java 17 - needs instanceof before switch
String format(Object obj) {
    if (obj instanceof Integer i) {
        return "Integer: " + i;
    } else if (obj instanceof String s) {
        return "String: " + s;
    } else if (obj == null) {
        return "Null";
    }
    return "Unknown";
}
Java 21 STABLE
// Java 21 - pattern matching directly in switch
String format(Object obj) {
    return switch (obj) {
        case Integer i -> "Integer: " + i;
        case String s -> "String: " + s;
        case null -> "Null";
        default -> "Unknown";
    };
}

// With guards (when clause)
String describe(Object obj) {
    return switch (obj) {
        case Integer i when i > 0 -> "Positive";
        case Integer i when i < 0 -> "Negative";
        case Integer i -> "Zero";
        case String s when s.length() > 10 -> "Long string";
        case String s -> "Short string";
        default -> "Other";
    };
}
Java 21 advantage: Cleaner and more readable code, integrated pattern matching in switch.

Record Patterns

Java 17 → Records without destructuring
Java 21 → Record patterns (stable)
Java 17
record Point(int x, int y) {}
record Rectangle(Point topLeft, Point bottomRight) {}

// Manual decomposition
void print(Rectangle r) {
    if (r instanceof Rectangle) {
        Point p1 = r.topLeft();
        Point p2 = r.bottomRight();
        System.out.println(p1.x() + "," + p1.y());
    }
}
Java 21 STABLE
record Point(int x, int y) {}
record Rectangle(Point topLeft, Point bottomRight) {}

// Automatic destructuring
void print(Rectangle r) {
    if (r instanceof Rectangle(Point(int x1, int y1), 
                                Point(int x2, int y2))) {
        System.out.println(x1 + "," + y1 + " - " + x2 + "," + y2);
    }
}

// In switch
int area(Object obj) {
    return switch (obj) {
        case Rectangle(Point(int x1, int y1), 
                       Point(int x2, int y2)) -> (x2 - x1) * (y2 - y1);
        default -> 0;
    };
}
Java 21 advantage: Automatic Record destructuring, more expressive code.

Sequenced Collections

Java 17 → Separate interfaces for ordered lists
Java 21 → New SequencedCollection interface
import java.util.*;

// Java 21 - New interfaces for ordered collections
// SequencedCollection, SequencedSet, SequencedMap

public class SequencedCollectionsDemo {
    public static void main(String[] args) {
        
        // ========== SEQUENCED COLLECTION ==========
        SequencedCollection<String> list = new ArrayList<>(List.of("A", "B", "C"));
        
        // New methods
        String first = list.getFirst();     // "A"
        String last = list.getLast();       // "C"
        
        list.addFirst("Z");    // [Z, A, B, C]
        list.addLast("D");     // [Z, A, B, C, D]
        
        list.removeFirst();    // remove "Z"
        list.removeLast();     // remove "D"
        
        // Reverse order view
        SequencedCollection<String> reversed = list.reversed();
        System.out.println("Reversed: " + reversed);  // [C, B, A]
        
        // ========== SEQUENCED MAP ==========
        SequencedMap<String, Integer> map = new LinkedHashMap<>();
        map.put("A", 1);
        map.put("B", 2);
        map.put("C", 3);
        
        // First and last entry
        Map.Entry<String, Integer> firstEntry = map.firstEntry();    // A=1
        Map.Entry<String, Integer> lastEntry = map.lastEntry();      // C=3
        
        // Reversed map
        SequencedMap<String, Integer> reversedMap = map.reversed();
        
        // ========== ADVANTAGES ==========
        // Before (Java 17) to get first and last element:
        // List: list.get(0), list.get(list.size()-1)
        // Now: list.getFirst(), list.getLast()
    }
}
Output:
Reversed: [C, B, A]
First entry: A=1
Last entry: C=3

Virtual Threads (Project Loom)

Java 17 → Platform Threads (heavy)
Java 21 → Virtual Threads (lightweight) STABLE
import java.util.concurrent.*;
import java.util.stream.IntStream;

public class VirtualThreadsDemo {
    public static void main(String[] args) throws Exception {
        
        // ========== PLATFORM THREADS (Java 17) ==========
        System.out.println("=== Platform Threads (heavy) ===");
        long startPlatform = System.currentTimeMillis();
        
        try (ExecutorService executor = Executors.newFixedThreadPool(100)) {
            IntStream.range(0, 1000).forEach(i -> {
                executor.submit(() -> {
                    try { Thread.sleep(10); } catch (InterruptedException e) {}
                });
            });
            executor.shutdown();
            executor.awaitTermination(1, TimeUnit.MINUTES);
        }
        
        long endPlatform = System.currentTimeMillis();
        System.out.println("Platform threads: " + (endPlatform - startPlatform) + "ms");
        
        // ========== VIRTUAL THREADS (Java 21) ==========
        System.out.println("\n=== Virtual Threads (lightweight) ===");
        long startVirtual = System.currentTimeMillis();
        
        try (ExecutorService executor = Executors.newVirtualThreadPerTaskExecutor()) {
            IntStream.range(0, 1_000_000).forEach(i -> {
                executor.submit(() -> {
                    try { Thread.sleep(10); } catch (InterruptedException e) {}
                });
            });
            executor.shutdown();
            executor.awaitTermination(1, TimeUnit.MINUTES);
        }
        
        long endVirtual = System.currentTimeMillis();
        System.out.println("Virtual threads: " + (endVirtual - startVirtual) + "ms");
        
        // ========== CREATING VIRTUAL THREADS ==========
        System.out.println("\n=== Creating Virtual Threads ===");
        
        // Method 1: Thread.ofVirtual()
        Thread vt1 = Thread.ofVirtual()
            .name("virtual-thread-1")
            .start(() -> System.out.println("Hello from virtual thread"));
        
        // Method 2: Thread.startVirtualThread()
        Thread vt2 = Thread.startVirtualThread(() -> 
            System.out.println("Hello from another virtual thread"));
        
        // Method 3: Executors.newVirtualThreadPerTaskExecutor()
        try (ExecutorService executor = Executors.newVirtualThreadPerTaskExecutor()) {
            executor.submit(() -> System.out.println("Task on virtual thread"));
        }
        
        vt1.join();
        vt2.join();
        
        // ========== PERFORMANCE COMPARISON ==========
        System.out.println("\n=== Performance comparison ===");
        System.out.println("Platform threads: ~1000 threads max");
        System.out.println("Virtual threads: millions of simultaneous threads!");
    }
}
Output:
Platform threads: ~250ms (with 1000 threads)
Virtual threads: ~500ms (with 1,000,000 threads!)
Virtual threads are extremely lightweight and allow millions of simultaneous threads.

Structured Concurrency (Preview)

Java 17 → Manual concurrency management
Java 21 → Structured Concurrency (Preview)
import java.util.concurrent.*;
import java.util.concurrent.StructuredTaskScope.*;

// Java 21 - StructuredTaskScope (Preview)
// Requires: --enable-preview

public class StructuredConcurrencyDemo {
    
    // ========== WITHOUT STRUCTURED CONCURRENCY ==========
    static class UserProfile {
        String userId, name, orders, reviews;
    }
    
    UserProfile fetchProfileLegacy(String userId) throws Exception {
        // With CompletableFuture (unstructured)
        CompletableFuture<String> nameFuture = CompletableFuture.supplyAsync(() -> 
            fetchName(userId));
        CompletableFuture<String> ordersFuture = CompletableFuture.supplyAsync(() -> 
            fetchOrders(userId));
        CompletableFuture<String> reviewsFuture = CompletableFuture.supplyAsync(() -> 
            fetchReviews(userId));
        
        // Waits for all, but if one fails, others continue
        CompletableFuture.allOf(nameFuture, ordersFuture, reviewsFuture).join();
        
        UserProfile profile = new UserProfile();
        profile.name = nameFuture.get();
        profile.orders = ordersFuture.get();
        profile.reviews = reviewsFuture.get();
        return profile;
    }
    
    // ========== WITH STRUCTURED CONCURRENCY (Java 21) ==========
    UserProfile fetchProfileStructured(String userId) throws Exception {
        // Structured scope: all subtasks are managed together
        try (var scope = new StructuredTaskScope.ShutdownOnFailure()) {
            
            Subtask<String> nameTask = scope.fork(() -> fetchName(userId));
            Subtask<String> ordersTask = scope.fork(() -> fetchOrders(userId));
            Subtask<String> reviewsTask = scope.fork(() -> fetchReviews(userId));
            
            // If one subtask fails, all are cancelled
            scope.join();           // Waits for all
            scope.throwIfFailed();  // Propagates any exceptions
            
            UserProfile profile = new UserProfile();
            profile.name = nameTask.get();
            profile.orders = ordersTask.get();
            profile.reviews = reviewsTask.get();
            return profile;
        }
        // On exit from try, all threads are automatically terminated
    }
    
    private String fetchName(String userId) {
        return "Mario Rossi";
    }
    
    private String fetchOrders(String userId) {
        return "5 orders";
    }
    
    private String fetchReviews(String userId) {
        return "10 reviews";
    }
    
    public static void main(String[] args) throws Exception {
        StructuredConcurrencyDemo demo = new StructuredConcurrencyDemo();
        UserProfile profile = demo.fetchProfileStructured("123");
        System.out.println("Profile: " + profile.name + ", " + profile.orders);
    }
}
Output:
Profile: Mario Rossi, 5 orders

String Templates (Preview)

Java 17 → Concatenation or String.format()
Java 21 → String Templates (Preview)
// Java 21 - String Templates (Preview)
// Requires: --enable-preview

public class StringTemplatesDemo {
    public static void main(String[] args) {
        String name = "Mario";
        int age = 30;
        double price = 49.99;
        
        // ========== OLD METHODS ==========
        System.out.println("=== Old methods ===");
        
        // Concatenation
        String msg1 = "User: " + name + ", age: " + age;
        
        // String.format()
        String msg2 = String.format("User: %s, age: %d", name, age);
        
        // ========== STRING TEMPLATES (Java 21) ==========
        System.out.println("\n=== String Templates (Java 21) ===");
        
        // Simple template
        String msg3 = STR."User: \{name}, age: \{age}";
        System.out.println(msg3);
        
        // Expressions in templates
        String msg4 = STR."In one year \{name} will be \{age + 1} years old";
        System.out.println(msg4);
        
        // Multiline template
        String json = STR."""
            {
                "name": "\{name}",
                "age": \{age},
                "price": \{price}
            }
            """;
        System.out.println(json);
        
        // Template with custom processor
        // FMT processor for formatting
        String formatted = FMT."Price: %.2f\{price}";
        System.out.println(formatted);
        
        // ========== CUSTOM PROCESSOR ==========
        // JSON processor
        var JSON = StringTemplate.Processor.of((StringTemplate st) -> {
            // JSON validation and transformation
            return st.interpolate();
        });
        
        // SQL processor (prevents SQL injection)
        var SQL = StringTemplate.Processor.of((StringTemplate st) -> {
            // Automatic sanitization
            return st.interpolate();
        });
    }
}
Output:
User: Mario, age: 30
In one year Mario will be 31 years old
{"name": "Mario", "age": 30, "price": 49.99}

Unnamed Variables

Java 17 → Required named variables
Java 21 → Unnamed variables (_) STABLE
import java.util.*;
import java.util.function.*;

// Java 21 - Unnamed variables ( _ )

public class UnnamedVariablesDemo {
    public static void main(String[] args) {
        
        // ========== UNUSED VARIABLES ==========
        // Java 17 - named variable
        try {
            int result = 10 / 0;
        } catch (ArithmeticException e) {
            // 'e' is not used
        }
        
        // Java 21 - unnamed variable
        try {
            int result = 10 / 0;
        } catch (ArithmeticException _) {
            // Variable is anonymous, clear intention
            System.out.println("Division by zero");
        }
        
        // ========== LAMBDA EXPRESSIONS ==========
        List<String> list = Arrays.asList("a", "b", "c");
        
        // Java 17 - named parameter
        list.forEach(item -> System.out.println("Element"));
        
        // Java 21 - unnamed parameter
        list.forEach(_ -> System.out.println("Element"));
        
        // ========== TRY-WITH-RESOURCES ==========
        // Java 17
        try (FileReader reader = new FileReader("file.txt")) {
            // reader not used
        } catch (IOException e) {
            // e not used
        }
        
        // Java 21
        try (FileReader _ = new FileReader("file.txt")) {
            // Anonymous resource
        } catch (IOException _) {
            // Anonymous exception
        }
        
        // ========== PATTERN MATCHING ==========
        Object obj = "test";
        
        // Java 21 - unnamed in pattern matching
        if (obj instanceof String _) {
            System.out.println("It's a string");
        }
        
        // In record patterns
        record Point(int x, int y) {}
        Point p = new Point(5, 10);
        
        if (p instanceof Point(int x, int _)) {
            System.out.println("x = " + x);
        }
    }
}
Output:
Division by zero
Element
It's a string
x = 5

Scoped Values (Preview)

Java 17 → ThreadLocal (mutable, expensive)
Java 21 → Scoped Values (immutable, efficient)
import java.util.concurrent.*;

// Java 21 - Scoped Values (Preview)
// More efficient alternative to ThreadLocal

public class ScopedValuesDemo {
    
    // ========== THREAD LOCAL (Java 17) ==========
    private static final ThreadLocal<String> threadLocal = new ThreadLocal<>();
    
    static void threadLocalExample() {
        threadLocal.set("value");
        String value = threadLocal.get();
        threadLocal.remove();  // Necessary to avoid memory leaks
    }
    
    // ========== SCOPED VALUES (Java 21) ==========
    private static final ScopedValue<String> SCOPED = ScopedValue.newInstance();
    
    static void scopedValueExample() {
        // Immutable binding
        ScopedValue.where(SCOPED, "value").run(() -> {
            // Inside this scope, SCOPED.get() returns "value"
            String value = SCOPED.get();
            System.out.println("Value: " + value);
            
            // Nested operations
            nestedOperation();
        });
        // Outside the scope, SCOPED is not defined
    }
    
    static void nestedOperation() {
        // Inherits the value from the parent scope
        String value = SCOPED.get();
        System.out.println("Nested: " + value);
    }
    
    // ========== CONCURRENCY ==========
    static void concurrentExample() throws Exception {
        try (var executor = Executors.newVirtualThreadPerTaskExecutor()) {
            for (int i = 0; i < 10; i++) {
                final int id = i;
                executor.submit(() -> {
                    ScopedValue.where(SCOPED, "User-" + id).run(() -> {
                        processRequest();
                    });
                });
            }
        }
    }
    
    static void processRequest() {
        String userId = SCOPED.get();
        System.out.println("Processing request for: " + userId);
    }
    
    public static void main(String[] args) throws Exception {
        scopedValueExample();
        concurrentExample();
    }
}
Output:
Value: value
Nested: value
Processing request for: User-0
Processing request for: User-1

New APIs

New APIs introduced in Java 21
import java.util.*;
import java.util.concurrent.*;
import java.time.*;

public class NewJava21APIs {
    public static void main(String[] args) {
        
        // ========== 1. SEQUENCED COLLECTIONS ==========
        SequencedCollection<String> list = new ArrayList<>(List.of("A", "B", "C"));
        System.out.println("First: " + list.getFirst());  // A
        System.out.println("Last: " + list.getLast());    // C
        
        // ========== 2. MATH API ==========
        // Division with rounding toward zero
        long result1 = Math.divideExact(10, 3);   // 3
        // Clamping - limits a value between min and max
        int clamped = Math.clamp(15, 10, 20);     // 15
        int clamped2 = Math.clamp(5, 10, 20);     // 10
        int clamped3 = Math.clamp(25, 10, 20);    // 20
        
        // ========== 3. COLLECTION FACTORY METHODS ==========
        // Set.of, List.of, Map.of already existing, now more efficient
        
        // ========== 4. FUTURE TASKS ==========
        // New methods in Future
        Future<String> future = CompletableFuture.completedFuture("done");
        // future.resultNow() - gets result if completed
        // future.exceptionNow() - gets exception if failed
        // future.state() - gets state (RUNNING, SUCCESS, FAILED)
        
        // ========== 5. TIME API ==========
        LocalDate date = LocalDate.of(2024, 10, 15);
        DayOfWeek dow = DayOfWeek.from(date);
        // New formatting methods
        String formatted = date.format(java.time.format.DateTimeFormatter.ISO_LOCAL_DATE);
        
        // ========== 6. ZONED DATE TIME ==========
        ZonedDateTime now = ZonedDateTime.now();
        // Methods for time zone conversion
        ZonedDateTime utc = now.withZoneSameInstant(ZoneId.of("UTC"));
        
        System.out.println("Clamped: " + clamped);
        System.out.println("Formatted date: " + formatted);
    }
}
Output:
First: A
Last: C
Clamped: 15
Formatted date: 2024-10-15
