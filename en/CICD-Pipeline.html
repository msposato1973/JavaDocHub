    CI/CD Glossary

    Continuous Integration (CI)

    Practice of frequently integrating code into a shared repository with automated builds and tests.

    Continuous Delivery (CD)

    Extension of CI: the software is always ready for production deployment.

    Continuous Deployment

    Every change that passes tests is automatically deployed to production.

    Pipeline

    Automated sequence of steps that code goes through from commit to deploy.

    Artifact

    Build output (JAR, Docker image, bundle) ready for deployment.

    Orchestrator

    Tool that manages pipeline execution (Jenkins, GitLab CI, GitHub Actions).
