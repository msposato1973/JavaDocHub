Object-Oriented Programming in Java

Classes, objects, encapsulation, inheritance, polymorphism, abstraction, and interfaces

What is Object-Oriented Programming?

Object-Oriented Programming (OOP) is a programming paradigm that organizes code into "objects" that contain data (attributes) and behaviors (methods). The four fundamental pillars of OOP are:
PillarDescriptionBenefit
Encapsulation Hide internal details, expose only what is needed Security and maintainability
Inheritance Reuse code from existing classes Reusability and hierarchies
Polymorphism An object can take many forms Flexibility and extensibility
Abstraction Hide complexity, show only the essentials Simplicity and clarity

Classes and Objects

The heart of OOP programming
A class is a blueprint for creating objects. An object is an instance of a class, with state (attributes) and behavior (methods). 
// Class definition
public class Car {
    // Attributes (state)
    private String brand;
    private String model;
    private int year;
    private double speed;
    
    // Constructor
    public Car(String brand, String model, int year) {
        this.brand = brand;
        this.model = model;
        this.year = year;
        this.speed = 0;
    }
    
    // Methods (behavior)
    public void accelerate(double increment) {
        speed += increment;
        System.out.println(brand + " " + model + " accelerates to " + speed + " km/h");
    }
    
    public void brake(double decrement) {
        speed = Math.max(0, speed - decrement);
        System.out.println(brand + " " + model + " brakes to " + speed + " km/h");
    }
    
    public void printInfo() {
        System.out.println("Car: " + brand + " " + model + " (" + year + ")");
    }
    
    // Getters and Setters
    public String getBrand() { return brand; }
    public String getModel() { return model; }
    public int getYear() { return year; }
    public double getSpeed() { return speed; }
}

// Using objects
public class Main {
    public static void main(String[] args) {
        // Creating objects
        Car car1 = new Car("Fiat", "Panda", 2020);
        Car car2 = new Car("Tesla", "Model 3", 2023);
        
        car1.printInfo();
        car1.accelerate(50);
        car1.brake(20);
        
        car2.printInfo();
        car2.accelerate(80);
        
        System.out.println("Car1 speed: " + car1.getSpeed() + " km/h");
    }
}
Output:
Car: Fiat Panda (2020)
Fiat Panda accelerates to 50.0 km/h
Fiat Panda brakes to 30.0 km/h
Car: Tesla Model 3 (2023)
Tesla Model 3 accelerates to 80.0 km/h
Car1 speed: 30.0 km/h
Key concepts:
- new creates a new object in memory
- this refers to the current object
- Each object has its own copy of attributes

Encapsulation

Hide implementation details, expose only what is needed
Encapsulation protects a class's internal data, allowing access only through public methods (getters and setters). This ensures control over modifications and maintains data integrity.
❌ WITHOUT ENCAPSULATION
public class BankAccount {
    public double balance;  // Direct access - dangerous!
}

BankAccount account = new BankAccount();
account.balance = -100;  // Negative balance possible!
✅ WITH ENCAPSULATION
public class BankAccount {
    private double balance;  // Access only via methods
    
    public void deposit(double amount) {
        if (amount > 0) balance += amount;
    }
    
    public boolean withdraw(double amount) {
        if (amount > 0 && amount <= balance) {
            balance -= amount;
            return true;
        }
        return false;
    }
    
    public double getBalance() { return balance; }
}
public class Person {
    // Private attributes
    private String name;
    private int age;
    private String email;
    
    // Constructor
    public Person(String name, int age, String email) {
        this.name = name;
        setAge(age);  // Uses setter for validation
        setEmail(email);
    }
    
    // Getters (controlled access)
    public String getName() { return name; }
    public int getAge() { return age; }
    public String getEmail() { return email; }
    
    // Setters with validation
    public void setName(String name) {
        if (name != null && !name.isEmpty()) {
            this.name = name;
        }
    }
    
    public void setAge(int age) {
        if (age >= 0 && age <= 150) {
            this.age = age;
        } else {
            throw new IllegalArgumentException("Invalid age: " + age);
        }
    }
    
    public void setEmail(String email) {
        if (email != null && email.contains("@")) {
            this.email = email;
        } else {
            throw new IllegalArgumentException("Invalid email");
        }
    }
    
    public void printInfo() {
        System.out.println("Name: " + name + ", Age: " + age + ", Email: " + email);
    }
}
Output:
Person p = new Person("Mario", 30, "mario@email.com");
p.printInfo(); // Name: Mario, Age: 30, Email: mario@email.com
p.setAge(31);
System.out.println("New age: " + p.getAge()); // New age: 31
Benefits of encapsulation:
- Data protection (no direct access)
- Input validation (in setters)
- Flexibility (can change internal implementation without impacting clients)
- Code maintainability

Inheritance

Code reuse through class hierarchies
Inheritance allows a class (subclass) to inherit attributes and methods from another class (superclass), promoting code reuse and establishing "is-a" relationships. 
// Base class (superclass)
class Animal {
    protected String name;
    protected int age;
    
    public Animal(String name, int age) {
        this.name = name;
        this.age = age;
    }
    
    public void eat() {
        System.out.println(name + " is eating");
    }
    
    public void sleep() {
        System.out.println(name + " is sleeping");
    }
    
    public void makeSound() {
        System.out.println(name + " makes a generic sound");
    }
    
    public void printInfo() {
        System.out.println("Animal: " + name + ", Age: " + age);
    }
}

// Subclass Dog
class Dog extends Animal {
    private String breed;
    
    public Dog(String name, int age, String breed) {
        super(name, age);  // Call to superclass constructor
        this.breed = breed;
    }
    
    @Override
    public void makeSound() {
        System.out.println(name + " barks: Woof Woof!");
    }
    
    public void run() {
        System.out.println(name + " is running happily!");
    }
    
    public void setBreed(String breed) { this.breed = breed; }
    public String getBreed() { return breed; }
}

// Subclass Cat
class Cat extends Animal {
    private String color;
    
    public Cat(String name, int age, String color) {
        super(name, age);
        this.color = color;
    }
    
    @Override
    public void makeSound() {
        System.out.println(name + " meows: Meow Meow!");
    }
    
    public void scratch() {
        System.out.println(name + " scratches the sofa!");
    }
    
    public void setColor(String color) { this.color = color; }
    public String getColor() { return color; }
}

// Subclass Fish
class Fish extends Animal {
    public Fish(String name, int age) {
        super(name, age);
    }
    
    @Override
    public void makeSound() {
        System.out.println(name + " blows bubbles...");
    }
    
    public void swim() {
        System.out.println(name + " is swimming!");
    }
}

public class InheritanceDemo {
    public static void main(String[] args) {
        Dog dog = new Dog("Buddy", 3, "Labrador");
        Cat cat = new Cat("Luna", 2, "Black");
        Fish fish = new Fish("Nemo", 1);
        
        // Inherited methods
        dog.eat();      // Inherited from Animal
        cat.sleep();    // Inherited from Animal
        fish.printInfo(); // Inherited from Animal
        
        // Overridden methods
        dog.makeSound();    // Override in Dog
        cat.makeSound();    // Override in Cat
        fish.makeSound();   // Override in Fish
        
        // Specific methods
        dog.run();
        cat.scratch();
        fish.swim();
        
        System.out.println("Breed of " + dog.name + ": " + dog.getBreed());
        System.out.println("Color of " + cat.name + ": " + cat.getColor());
    }
}
Output:
Buddy is eating
Luna is sleeping
Animal: Nemo, Age: 1
Buddy barks: Woof Woof!
Luna meows: Meow Meow!
Nemo blows bubbles...
Buddy is running happily!
Luna scratches the sofa!
Nemo is swimming!
Breed of Buddy: Labrador
Color of Luna: Black
Key inheritance concepts:
- Use extends to inherit from a class
- super() calls the superclass constructor
- @Override indicates you are overriding a method
- Java supports single inheritance (one superclass only)
- protected allows access to subclasses

Polymorphism

An object can take many forms
Polymorphism allows objects of different classes to respond to the same message (method) in different ways. There are two types: compile-time polymorphism (overloading) and runtime polymorphism (overriding).

📌 Overloading (Compile-time polymorphism)

class Calculator {
    // Same name, different parameters
    public int sum(int a, int b) {
        return a + b;
    }
    
    public int sum(int a, int b, int c) {
        return a + b + c;
    }
    
    public double sum(double a, double b) {
        return a + b;
    }
    
    public String sum(String a, String b) {
        return a + b;
    }
}

// Usage
Calculator calc = new Calculator();
System.out.println(calc.sum(5, 3));       // 8
System.out.println(calc.sum(5, 3, 2));    // 10
System.out.println(calc.sum(5.5, 3.2));   // 8.7
System.out.println(calc.sum("Hello ", "World")); // "Hello World"

📌 Overriding (Runtime polymorphism)

// Common interface
interface Shape {
    double calculateArea();
    void printInfo();
}

class Rectangle implements Shape {
    private double width, height;
    
    public Rectangle(double width, double height) {
        this.width = width;
        this.height = height;
    }
    
    @Override
    public double calculateArea() {
        return width * height;
    }
    
    @Override
    public void printInfo() {
        System.out.println("Rectangle: width=" + width + ", height=" + height);
    }
}

class Circle implements Shape {
    private double radius;
    
    public Circle(double radius) {
        this.radius = radius;
    }
    
    @Override
    public double calculateArea() {
        return Math.PI * radius * radius;
    }
    
    @Override
    public void printInfo() {
        System.out.println("Circle: radius=" + radius);
    }
}

class Triangle implements Shape {
    private double base, height;
    
    public Triangle(double base, double height) {
        this.base = base;
        this.height = height;
    }
    
    @Override
    public double calculateArea() {
        return (base * height) / 2;
    }
    
    @Override
    public void printInfo() {
        System.out.println("Triangle: base=" + base + ", height=" + height);
    }
}

public class PolymorphismDemo {
    public static void main(String[] args) {
        // Polymorphism: treating different objects uniformly
        Shape[] shapes = {
            new Rectangle(5, 3),
            new Circle(4),
            new Triangle(6, 4)
        };
        
        System.out.println("--- Areas of shapes ---");
        for (Shape shape : shapes) {
            shape.printInfo();
            System.out.println("Area: " + String.format("%.2f", shape.calculateArea()));
            System.out.println();
        }
        
        // Polymorphism with abstract classes
        Animal[] animals = {
            new Dog("Buddy", 3, "Labrador"),
            new Cat("Luna", 2, "Black"),
            new Fish("Nemo", 1)
        };
        
        System.out.println("--- Animal sounds ---");
        for (Animal animal : animals) {
            animal.makeSound();  // Polymorphic behavior!
        }
    }
}

Abstraction and Abstract Classes

Define contracts without implementation
Abstraction allows defining methods without implementation (abstract methods) that must be implemented by subclasses. Abstract classes cannot be instantiated directly. 
// Abstract class - cannot be instantiated
abstract class Employee {
    protected String firstName;
    protected String lastName;
    protected String employeeId;
    
    public Employee(String firstName, String lastName, String employeeId) {
        this.firstName = firstName;
        this.lastName = lastName;
        this.employeeId = employeeId;
    }
    
    // Concrete method (implemented)
    public void printBasicInfo() {
        System.out.println("Employee: " + firstName + " " + lastName);
        System.out.println("ID: " + employeeId);
    }
    
    // Abstract method - must be implemented by subclasses
    public abstract double calculateSalary();
    
    // Abstract method
    public abstract String getRole();
}

// Concrete subclass - Developer
class Developer extends Employee {
    private double baseSalary;
    private int overtimeHours;
    private final double OVERTIME_RATE = 25.0;
    
    public Developer(String firstName, String lastName, String employeeId, double baseSalary) {
        super(firstName, lastName, employeeId);
        this.baseSalary = baseSalary;
    }
    
    public void setOvertimeHours(int hours) {
        this.overtimeHours = hours;
    }
    
    @Override
    public double calculateSalary() {
        return baseSalary + (overtimeHours * OVERTIME_RATE);
    }
    
    @Override
    public String getRole() {
        return "Software Developer";
    }
}

// Concrete subclass - Manager
class Manager extends Employee {
    private double baseSalary;
    private double bonus;
    private double goalsPercentage;
    
    public Manager(String firstName, String lastName, String employeeId, double baseSalary) {
        super(firstName, lastName, employeeId);
        this.baseSalary = baseSalary;
    }
    
    public void setBonus(double bonus) {
        this.bonus = bonus;
    }
    
    public void setGoalsPercentage(double percentage) {
        this.goalsPercentage = percentage;
    }
    
    @Override
    public double calculateSalary() {
        double goalsBonus = baseSalary * (goalsPercentage / 100);
        return baseSalary + bonus + goalsBonus;
    }
    
    @Override
    public String getRole() {
        return "Manager";
    }
}

// Concrete subclass - Salesperson
class Salesperson extends Employee {
    private double baseSalary;
    private double commissions;
    
    public Salesperson(String firstName, String lastName, String employeeId, double baseSalary) {
        super(firstName, lastName, employeeId);
        this.baseSalary = baseSalary;
    }
    
    public void setCommissions(double commissions) {
        this.commissions = commissions;
    }
    
    @Override
    public double calculateSalary() {
        return baseSalary + commissions;
    }
    
    @Override
    public String getRole() {
        return "Salesperson";
    }
}

public class AbstractionDemo {
    public static void main(String[] args) {
        Developer dev = new Developer("Mario", "Rossi", "DEV001", 2500);
        dev.setOvertimeHours(10);
        
        Manager mgr = new Manager("Laura", "Bianchi", "MGR001", 4000);
        mgr.setBonus(1500);
        mgr.setGoalsPercentage(10);
        
        Salesperson sales = new Salesperson("Giuseppe", "Verdi", "SAL001", 1200);
        sales.setCommissions(3500);
        
        // Polymorphism with abstract class
        Employee[] employees = {dev, mgr, sales};
        
        System.out.println("=== EMPLOYEE LIST ===\n");
        for (Employee e : employees) {
            e.printBasicInfo();
            System.out.println("Role: " + e.getRole());
            System.out.printf("Salary: €%.2f\n", e.calculateSalary());
            System.out.println("---");
        }
    }
}

Abstract class vs Interface:
- Abstract class: can have state (attributes) and concrete methods
- Interface: only abstract methods (until Java 7), default/static from Java 8
- A class can extend ONLY ONE abstract class
- A class can implement MULTIPLE interfaces

Interfaces

Contracts that define "what to do" not "how to do it"
An interface defines a contract that classes must respect. Unlike abstract classes, a class can implement multiple interfaces (multiple inheritance). 
// Interface definitions
interface Flyable {
    void fly();
    
    // Default methods (Java 8+)
    default void glide() {
        System.out.println("Gliding...");
    }
    
    // Static methods (Java 8+)
    static void takeOff() {
        System.out.println("Taking off...");
    }
}

interface Swimmable {
    void swim();
}

interface Predator {
    void hunt();
    
    default void spotPrey() {
        System.out.println("Prey spotted!");
    }
}

interface Reproducible {
    void reproduce();
}

// Class implementing multiple interfaces
class Eagle implements Flyable, Predator, Reproducible {
    private String name;
    
    public Eagle(String name) {
        this.name = name;
    }
    
    @Override
    public void fly() {
        System.out.println(name + " flies majestically in the sky!");
    }
    
    @Override
    public void hunt() {
        System.out.println(name + " dives down to catch prey!");
    }
    
    @Override
    public void reproduce() {
        System.out.println(name + " builds a nest and lays eggs.");
    }
}

class Penguin implements Flyable, Swimmable {
    private String name;
    
    public Penguin(String name) {
        this.name = name;
    }
    
    @Override
    public void fly() {
        System.out.println(name + " cannot fly, but swims beautifully!");
    }
    
    @Override
    public void swim() {
        System.out.println(name + " swims rapidly in the water!");
    }
}

class Shark implements Swimmable, Predator {
    private String name;
    
    public Shark(String name) {
        this.name = name;
    }
    
    @Override
    public void swim() {
        System.out.println(name + " swims menacingly!");
    }
    
    @Override
    public void hunt() {
        System.out.println(name + " hunts for prey!");
    }
}

// Functional interface (single abstract method) - for lambdas
@FunctionalInterface
interface MathOperation {
    int execute(int a, int b);
}

// Interface with multiple inheritance between interfaces
interface MarineAnimal extends Swimmable, Reproducible {
    void breatheUnderwater();
}

class Dolphin implements MarineAnimal {
    private String name;
    
    public Dolphin(String name) {
        this.name = name;
    }
    
    @Override
    public void swim() {
        System.out.println(name + " swims elegantly!");
    }
    
    @Override
    public void reproduce() {
        System.out.println(name + " gives birth to a calf.");
    }
    
    @Override
    public void breatheUnderwater() {
        System.out.println(name + " surfaces to breathe.");
    }
}

public class InterfacesDemo {
    public static void main(String[] args) {
        System.out.println("=== INTERFACES EXAMPLES ===\n");
        
        Eagle eagle = new Eagle("Golden Eagle");
        Penguin penguin = new Penguin("Emperor Penguin");
        Shark shark = new Shark("Great White Shark");

        eagle.fly();

        System.out.println();
        penguin.fly();
        penguin.swim();
        
        System.out.println();
        shark.swim();
        shark.hunt();
        
        System.out.println();
        dolphin.swim();
        dolphin.reproduce();
        dolphin.breatheUnderwater();

Constructors

Object initialization
public class Student {
    private String firstName;
    private String lastName;
    private int studentId;
    private double gradeAverage;
    private static int idCounter = 1000;
    
    // Instance initializer block (executed before constructor)
    {
        System.out.println("Instance initializer block executed");
        this.studentId = ++idCounter;
    }
    
    // Static initializer block (executed once when class is loaded)
    static {
        System.out.println("Static block - Student class loading");
        idCounter = 1000;
    }
    
    // Constructor without parameters (default)
    public Student() {
        this("Unknown", "Unknown");
        System.out.println("Default constructor called");
    }
    
    // Constructor with parameters
    public Student(String firstName, String lastName) {
        this(firstName, lastName, 0.0);
        System.out.println("Constructor with 2 parameters called");
    }
    
    // Full constructor
    public Student(String firstName, String lastName, double gradeAverage) {
        this.firstName = firstName;
        this.lastName = lastName;
        this.gradeAverage = gradeAverage;
        System.out.println("Full constructor called");
    }
    
    // Copy constructor
    public Student(Student other) {
        this(other.firstName, other.lastName, other.gradeAverage);
        System.out.println("Copy constructor called");
    }
    
    // Private constructor (Singleton pattern)
    private Student(int specialId) {
        this.studentId = specialId;
        this.firstName = "Special";
        this.lastName = "Special";
    }
    
    public static Student createSpecialStudent() {
        return new Student(9999);
    }
    
    public void printInfo() {
        System.out.println("Student: " + firstName + " " + lastName);
        System.out.println("Student ID: " + studentId);
        System.out.println("Grade average: " + gradeAverage);
    }
    
    public static void main(String[] args) {
        System.out.println("--- Creating objects ---\n");
        
        Student s1 = new Student();
        System.out.println();
        
        Student s2 = new Student("Mario", "Rossi");
        System.out.println();
        
        Student s3 = new Student("Laura", "Bianchi", 28.5);
        System.out.println();
        
        Student s4 = new Student(s3);  // Copy constructor
        System.out.println();
        
        Student s5 = Student.createSpecialStudent();
        
        System.out.println("\n--- Student information ---\n");
        s1.printInfo();
        System.out.println();
        s2.printInfo();
        System.out.println();
        s3.printInfo();
        System.out.println();
        s4.printInfo();
        System.out.println();
        s5.printInfo();
    }
}

Static Members

Belong to the class, not to instances
public class Counter {
    // Instance variable (each object has its own copy)
    private int instanceCounter;
    
    // Static variable (single copy shared among all objects)
    private static int globalCounter = 0;
    
    // Static constant (public static final)
    public static final String CLASS_NAME = "Counter";
    public static final int MAX_VALUE = 100;
    
    // Constructor
    public Counter() {
        instanceCounter = 0;
        globalCounter++;
        System.out.println("Created object #" + globalCounter);
    }
    
    // Instance method
    public void incrementInstance() {
        if (instanceCounter < MAX_VALUE) {
            instanceCounter++;
        }
    }
    
    public int getInstanceCounter() {
        return instanceCounter;
    }
    
    // Static method (can only access static members)
    public static int getGlobalCounter() {
        return globalCounter;
    }
    
    public static String getClassName() {
        return CLASS_NAME;
    }
    
    public static void resetGlobalCounter() {
        globalCounter = 0;
    }
    
    // Example of static inner class (nested class)
    public static class Utility {
        public static void printInfo() {
            System.out.println("Class: " + CLASS_NAME);
            System.out.println("Max value: " + MAX_VALUE);
            System.out.println("Counters created: " + globalCounter);
        }
    }
}

public class StaticDemo {
    public static void main(String[] args) {
        // Access static members without creating objects
        System.out.println("Class name: " + Counter.getClassName());
        System.out.println("Max value: " + Counter.MAX_VALUE);
        
        // Creating objects
        Counter c1 = new Counter();
        Counter c2 = new Counter();
        Counter c3 = new Counter();
        
        // Operations on instances
        c1.incrementInstance();
        c1.incrementInstance();
        c2.incrementInstance();
        
        System.out.println("\n--- Instance counters ---");
        System.out.println("c1: " + c1.getInstanceCounter());
        System.out.println("c2: " + c2.getInstanceCounter());
        System.out.println("c3: " + c3.getInstanceCounter());
        
        // Access static members
        System.out.println("\n--- Static members ---");
        System.out.println("Total objects created: " + Counter.getGlobalCounter());
        
        // Static inner class
        System.out.println("\n--- Static inner class ---");
        Counter.Utility.printInfo();
        
        // Practical example: shared counter between threads
        System.out.println("\n--- Shared counter between threads ---");
        Runnable task = () -> {
            for (int i = 0; i < 100; i++) {
                Counter.getGlobalCounter();  // read only
            }
            System.out.println(Thread.currentThread().getName() + " completed");
        };
        
        Thread t1 = new Thread(task, "Thread-1");
        Thread t2 = new Thread(task, "Thread-2");
        t1.start();
        t2.start();
    }
}

Static member rules:
- Static members belong to the class, not to instances
- Access via Class.member (no object needed)
- Static methods can only access static members

import java.util.*;

// ==================== ABSTRACT CLASS ====================
abstract class LibraryMaterial {
    protected String title;
    protected String code;
    protected boolean available;
    protected LocalDate insertionDate;
    
    public LibraryMaterial(String title, String code) {
        this.title = title;
        this.code = code;
        this.available = true;
        this.insertionDate = LocalDate.now();
    }
    
    public abstract String getType();
    public abstract int getLoanDays();
    
    public void lend() {
        if (available) {
            available = false;
            System.out.println("📖 " + title + " has been lent");
        } else {
            System.out.println("❌ " + title + " is not available");
        }
    }
    
    public void returnItem() {
        available = true;
        System.out.println("📚 " + title + " has been returned");
    }
    
    public void printInfo() {
        System.out.println("[" + getType() + "] " + title + " (Code: " + code + ")");
        System.out.println("  Available: " + (available ? "✅ Yes" : "❌ No"));
        System.out.println("  Added on: " + insertionDate);
        System.out.println("  Loan days: " + getLoanDays());
    }
}

// ==================== LIBRARY CLASS ====================
class Library {
    private String name;
    private String address;
    private List materials;
    private Map registeredUsers;
    
    public Library(String name, String address) {
        this.name = name;
        this.address = address;
        this.materials = new ArrayList<>();
        this.registeredUsers = new HashMap<>();
    }
    
    public void addMaterial(LibraryMaterial material) {
        materials.add(material);
        System.out.println("✅ Added: " + material.title);
    }
    
    public void removeMaterial(String code) {
        materials.removeIf(m -> m.code.equals(code));
        System.out.println("🗑️ Removed material with code: " + code);
    }
    
    public LibraryMaterial findByCode(String code) {
        for (LibraryMaterial m : materials) {
            if (m.code.equals(code)) {
                return m;
            }
        }
        return null;
    }
    
    public List findByTitle(String title) {
        List results = new ArrayList<>();
        for (LibraryMaterial m : materials) {
            if (m.title.toLowerCase().contains(title.toLowerCase())) {
                results.add(m);
            }
        }
        return results;
    }
    
    public List getAvailableMaterials() {
        List available = new ArrayList<>();
        for (LibraryMaterial m : materials) {
            if (m.available) {
                available.add(m);
            }
        }
        return available;
    }
    
    public void printCatalog() {
        System.out.println("\n📚 LIBRARY CATALOG '" + name + "' 📚");
        System.out.println("Address: " + address);
        System.out.println("Total materials: " + materials.size());
        System.out.println("Available: " + getAvailableMaterials().size());
        System.out.println("\n--- Materials list ---");
        
        for (LibraryMaterial m : materials) {
            m.printInfo();
            System.out.println();
        }
    }
    
    public void registerUser(User user) {
        registeredUsers.put(user.getTaxCode(), user);
        System.out.println("✅ User registered: " + user.getFirstName() + " " + user.getLastName());
    }
}

// ==================== USER CLASS ====================
class User {
    private String firstName;
    private String lastName;
    private String taxCode;
    private String email;
    private List activeLoans;
    
    public User(String firstName, String lastName, String taxCode, String email) {
        this.firstName = firstName;
        this.lastName = lastName;
        this.taxCode = taxCode;
        this.email = email;
        this.activeLoans = new ArrayList<>();
    }
    
    public void borrow(LibraryMaterial material) {
        if (material.available) {
            material.lend();
            activeLoans.add(material);
        } else {
            System.out.println("❌ " + material.title + " is not available for loan");
        }
    }
    
    public void returnMaterial(LibraryMaterial material) {
        if (activeLoans.remove(material)) {
            material.returnItem();
        } else {
            System.out.println("❌ " + material.title + " is not on loan by this user");
        }
    }
    
    public void printLoans() {
        System.out.println("\n📋 Active loans of " + firstName + " " + lastName + ":");
        if (activeLoans.isEmpty()) {
            System.out.println("  No active loans");
        } else {
            for (LibraryMaterial m : activeLoans) {
                System.out.println("  - " + m.title + " (due: " + 
                    LocalDate.now().plusDays(m.getLoanDays()) + ")");
            }
        }
    }
    
    // Getters
    public String getFirstName() { return firstName; }
    public String getLastName() { return lastName; }
    public String getTaxCode() { return taxCode; }
    public String getEmail() { return email; }
}

public class LibraryDemo {
    public static void main(String[] args) {
        // Create library
        Library library = new Library("Central Library", "Via Roma 1, Milan");
        
        // Create materials
        Book book1 = new Book("The Name of the Rose", "B001", "Umberto Eco", 560, "Bompiani");
        Book book2 = new Book("1984", "B002", "George Orwell", 328, "Mondadori");
        Book book3 = new Book("Java Complete Guide", "B003", "Herbert Schildt", 1200, "McGraw-Hill");
        
        DVD dvd1 = new DVD("Inception", "D001", 148, "MP4", "Christopher Nolan");
        DVD dvd2 = new DVD("Interstellar", "D002", 169, "MKV", "Christopher Nolan");
        
        Magazine magazine1 = new Magazine("National Geographic", "M001", 45, "Monthly");
        
        // Add to library
        System.out.println("=== ADDING MATERIALS ===\n");
        library.addMaterial(book1);
        library.addMaterial(book2);
        library.addMaterial(book3);
        library.addMaterial(dvd1);
        library.addMaterial(dvd2);
        library.addMaterial(magazine1);
        
        // Print catalog
        library.printCatalog();
        
        // Create user
        User user = new User("Mario", "Rossi", "RSSMRA80A01H501U", "mario.rossi@email.com");
        library.registerUser(user);
        
        // Loan operations
        System.out.println("\n=== LOAN OPERATIONS ===\n");
        user.borrow(book1);
        user.borrow(dvd1);
        
        // Attempt to borrow already lent book
        user.borrow(book1);
        
        // Print active loans
        user.printLoans();
        
        // Return
        System.out.println("\n=== RETURN ===\n");
        user.returnMaterial(book1);
        
        // Print loans after return
        user.printLoans();
        
        // Search by title
        System.out.println("\n=== SEARCH BY TITLE ===\n");
        List results = library.findByTitle("Java");
        System.out.println("Results for 'Java':");
        for (LibraryMaterial m : results) {
            System.out.println("  - " + m.title + " (" + m.getType() + ")");
        }
        
        // Polymorphism with Digital interface
        System.out.println("\n=== DIGITAL MATERIALS ===\n");
        for (LibraryMaterial m : library.getAvailableMaterials()) {
            if (m instanceof Digital) {
                Digital d = (Digital) m;
                System.out.println("📱 " + m.title + " - Format: " + d.getFormat());
                d.download();
            }
        }
        
        // Polymorphism with OnSiteConsultable interface
        System.out.println("\n=== ON-SITE CONSULTATION ===\n");
        for (LibraryMaterial m : library.getAvailableMaterials()) {
            if (m instanceof OnSiteConsultable) {
                ((OnSiteConsultable) m).consult();
            }
        }
    }
}
