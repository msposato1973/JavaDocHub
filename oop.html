Programmazione Orientata agli Oggetti in Java

Classi, oggetti, incapsulamento, ereditarietà, polimorfismo, astrazione e interfacce

Cos'è la Programmazione Orientata agli Oggetti?

La Programmazione Orientata agli Oggetti (OOP) è un paradigma di programmazione che organizza il codice in "oggetti" che contengono dati (attributi) e comportamenti (metodi). I quattro pilastri fondamentali dell'OOP sono:
PilastroDescrizioneBeneficio
Incapsulamento Nascondere i dettagli interni, esporre solo ciò che serve Sicurezza e manutenibilità
Ereditarietà Riutilizzare codice da classi esistenti Riutilizzo e gerarchie
Polimorfismo Un oggetto può assumere molte forme Flessibilità e estensibilità
Astrazione Nascondere la complessità, mostrare solo l'essenziale Semplicità e chiarezza

Classi e Oggetti

Il cuore della programmazione OOP
Una classe è un modello (blueprint) per creare oggetti. Un oggetto è un'istanza di una classe, con stato (attributi) e comportamento (metodi). 
// Definizione di una classe
public class Automobile {
    // Attributi (stato)
    private String marca;
    private String modello;
    private int anno;
    private double velocita;
    
    // Costruttore
    public Automobile(String marca, String modello, int anno) {
        this.marca = marca;
        this.modello = modello;
        this.anno = anno;
        this.velocita = 0;
    }
    
    // Metodi (comportamento)
    public void accelera(double incremento) {
        velocita += incremento;
        System.out.println(marca + " " + modello + " accelera a " + velocita + " km/h");
    }
    
    public void frena(double decremento) {
        velocita = Math.max(0, velocita - decremento);
        System.out.println(marca + " " + modello + " frena a " + velocita + " km/h");
    }
    
    public void stampaInfo() {
        System.out.println("Auto: " + marca + " " + modello + " (" + anno + ")");
    }
    
    // Getter e Setter
    public String getMarca() { return marca; }
    public String getModello() { return modello; }
    public int getAnno() { return anno; }
    public double getVelocita() { return velocita; }
}

// Utilizzo degli oggetti
public class Main {
    public static void main(String[] args) {
        // Creazione oggetti
        Automobile auto1 = new Automobile("Fiat", "Panda", 2020);
        Automobile auto2 = new Automobile("Tesla", "Model 3", 2023);
        
        auto1.stampaInfo();
        auto1.accelera(50);
        auto1.frena(20);
        
        auto2.stampaInfo();
        auto2.accelera(80);
        
        System.out.println("Velocità auto1: " + auto1.getVelocita() + " km/h");
    }
}
Output:
Auto: Fiat Panda (2020)
Fiat Panda accelera a 50.0 km/h
Fiat Panda frena a 30.0 km/h
Auto: Tesla Model 3 (2023)
Tesla Model 3 accelera a 80.0 km/h
Velocità auto1: 30.0 km/h
Concetti chiave:
- new crea un nuovo oggetto in memoria
- this si riferisce all'oggetto corrente
- Ogni oggetto ha la propria copia degli attributi

Incapsulamento (Encapsulation)

Nascondere i dettagli implementativi, esporre solo ciò che serve
L'incapsulamento protegge i dati interni di una classe, permettendo l'accesso solo attraverso metodi pubblici (getter e setter). Questo garantisce controllo sulle modifiche e mantiene l'integrità dei dati.
❌ SENZA INCAPSULAMENTO
public class ContoBancario {
    public double saldo;  // Accesso diretto - pericoloso!
}

ContoBancario conto = new ContoBancario();
conto.saldo = -100;  // Saldo negativo possibile!
✅ CON INCAPSULAMENTO
public class ContoBancario {
    private double saldo;  // Accesso solo tramite metodi
    
    public void deposita(double importo) {
        if (importo > 0) saldo += importo;
    }
    
    public boolean preleva(double importo) {
        if (importo > 0 && importo <= saldo) {
            saldo -= importo;
            return true;
        }
        return false;
    }
    
    public double getSaldo() { return saldo; }
}
public class Persona {
    // Attributi privati
    private String nome;
    private int eta;
    private String email;
    
    // Costruttore
    public Persona(String nome, int eta, String email) {
        this.nome = nome;
        setEta(eta);  // Usa il setter per la validazione
        setEmail(email);
    }
    
    // Getter (accesso controllato)
    public String getNome() { return nome; }
    public int getEta() { return eta; }
    public String getEmail() { return email; }
    
    // Setter con validazione
    public void setNome(String nome) {
        if (nome != null && !nome.isEmpty()) {
            this.nome = nome;
        }
    }
    
    public void setEta(int eta) {
        if (eta >= 0 && eta <= 150) {
            this.eta = eta;
        } else {
            throw new IllegalArgumentException("Età non valida: " + eta);
        }
    }
    
    public void setEmail(String email) {
        if (email != null && email.contains("@")) {
            this.email = email;
        } else {
            throw new IllegalArgumentException("Email non valida");
        }
    }
    
    public void stampaInfo() {
        System.out.println("Nome: " + nome + ", Età: " + eta + ", Email: " + email);
    }
}
Output:
Persona p = new Persona("Mario", 30, "mario@email.com");
p.stampaInfo(); // Nome: Mario, Età: 30, Email: mario@email.com
p.setEta(31);
System.out.println("Nuova età: " + p.getEta()); // Nuova età: 31
Vantaggi dell'incapsulamento:
- Protezione dei dati (nessun accesso diretto)
- Validazione degli input (nei setter)
- Flessibilità (puoi cambiare implementazione interna senza impattare i client)
- Manutenibilità del codice

Ereditarietà (Inheritance)

Riutilizzo del codice attraverso gerarchie di classi
L'ereditarietà permette a una classe (sottoclasse) di ereditare attributi e metodi da un'altra classe (superclasse), promuovendo il riutilizzo del codice e stabilendo relazioni "è-un" (is-a). 
// Classe base (superclasse)
class Animale {
    protected String nome;
    protected int eta;
    
    public Animale(String nome, int eta) {
        this.nome = nome;
        this.eta = eta;
    }
    
    public void mangia() {
        System.out.println(nome + " sta mangiando");
    }
    
    public void dormi() {
        System.out.println(nome + " sta dormendo");
    }
    
    public void faiVerso() {
        System.out.println(nome + " fa un verso generico");
    }
    
    public void stampaInfo() {
        System.out.println("Animale: " + nome + ", Età: " + eta);
    }
}

// Sottoclasse Cane
class Cane extends Animale {
    private String razza;
    
    public Cane(String nome, int eta, String razza) {
        super(nome, eta);  // Chiamata al costruttore della superclasse
        this.razza = razza;
    }
    
    @Override
    public void faiVerso() {
        System.out.println(nome + " abbaia: Bau Bau!");
    }
    
    public void corri() {
        System.out.println(nome + " sta correndo felice!");
    }
    
    public void setRazza(String razza) { this.razza = razza; }
    public String getRazza() { return razza; }
}

// Sottoclasse Gatto
class Gatto extends Animale {
    private String colore;
    
    public Gatto(String nome, int eta, String colore) {
        super(nome, eta);
        this.colore = colore;
    }
    
    @Override
    public void faiVerso() {
        System.out.println(nome + " miagola: Miao Miao!");
    }
    
    public void faiLeGrinfie() {
        System.out.println(nome + " fa le grinfie sul divano!");
    }
    
    public void setColore(String colore) { this.colore = colore; }
    public String getColore() { return colore; }
}

// Sottoclasse Pesce
class Pesce extends Animale {
    public Pesce(String nome, int eta) {
        super(nome, eta);
    }
    
    @Override
    public void faiVerso() {
        System.out.println(nome + " fa bolle...");
    }
    
    public void nuota() {
        System.out.println(nome + " sta nuotando!");
    }
}

public class EreditarietaDemo {
    public static void main(String[] args) {
        Cane cane = new Cane("Buddy", 3, "Labrador");
        Gatto gatto = new Gatto("Luna", 2, "Nero");
        Pesce pesce = new Pesce("Nemo", 1);
        
        // Metodi ereditati
        cane.mangia();      // Ereditato da Animale
        gatto.dormi();      // Ereditato da Animale
        pesce.stampaInfo(); // Ereditato da Animale
        
        // Metodi override
        cane.faiVerso();    // Override in Cane
        gatto.faiVerso();   // Override in Gatto
        pesce.faiVerso();   // Override in Pesce
        
        // Metodi specifici
        cane.corri();
        gatto.faiLeGrinfie();
        pesce.nuota();
        
        System.out.println("Razza di " + cane.nome + ": " + cane.getRazza());
        System.out.println("Colore di " + gatto.nome + ": " + gatto.getColore());
    }
}

Concetti chiave sull'ereditarietà:
- Usa extends per ereditare da una classe
- super() chiama il costruttore della superclasse
- @Override indica che stai sovrascrivendo un metodo
- Java supporta ereditarietà singola (una sola superclasse)
- protected permette accesso alle sottoclassi

Polimorfismo (Polymorphism)

Un oggetto può assumere molte forme
Il polimorfismo permette a oggetti di classi diverse di rispondere allo stesso messaggio (metodo) in modi diversi. Esistono due tipi: polimorfismo a tempo di compilazione (overloading) e a tempo di esecuzione (overriding).

📌 Overloading (Compile-time polymorphism)

class Calcolatrice {
    // Stesso nome, parametri diversi
    public int somma(int a, int b) {
        return a + b;
    }
    
    public int somma(int a, int b, int c) {
        return a + b + c;
    }
    
    public double somma(double a, double b) {
        return a + b;
    }
    
    public String somma(String a, String b) {
        return a + b;
    }
}

// Utilizzo
Calcolatrice calc = new Calcolatrice();
System.out.println(calc.somma(5, 3));       // 8
System.out.println(calc.somma(5, 3, 2));    // 10
System.out.println(calc.somma(5.5, 3.2));   // 8.7
System.out.println(calc.somma("Ciao ", "Mondo")); // "Ciao Mondo"

📌 Overriding (Runtime polymorphism)

// Interfaccia comune
interface Forma {
    double calcolaArea();
    void stampaInfo();
}

class Rettangolo implements Forma {
    private double base, altezza;
    
    public Rettangolo(double base, double altezza) {
        this.base = base;
        this.altezza = altezza;
    }
    
    @Override
    public double calcolaArea() {
        return base * altezza;
    }
    
    @Override
    public void stampaInfo() {
        System.out.println("Rettangolo: base=" + base + ", altezza=" + altezza);
    }
}

class Cerchio implements Forma {
    private double raggio;
    
    public Cerchio(double raggio) {
        this.raggio = raggio;
    }
    
    @Override
    public double calcolaArea() {
        return Math.PI * raggio * raggio;
    }
    
    @Override
    public void stampaInfo() {
        System.out.println("Cerchio: raggio=" + raggio);
    }
}

class Triangolo implements Forma {
    private double base, altezza;
    
    public Triangolo(double base, double altezza) {
        this.base = base;
        this.altezza = altezza;
    }
    
    @Override
    public double calcolaArea() {
        return (base * altezza) / 2;
    }
    
    @Override
    public void stampaInfo() {
        System.out.println("Triangolo: base=" + base + ", altezza=" + altezza);
    }
}

public class PolimorfismoDemo {
    public static void main(String[] args) {
        // Polimorfismo: trattiamo oggetti diversi in modo uniforme
        Forma[] forme = {
            new Rettangolo(5, 3),
            new Cerchio(4),
            new Triangolo(6, 4)
        };
        
        System.out.println("--- Aree delle forme ---");
        for (Forma forma : forme) {
            forma.stampaInfo();
            System.out.println("Area: " + String.format("%.2f", forma.calcolaArea()));
            System.out.println();
        }
        
        // Polimorfismo con classi astratte
        Animale[] animali = {
            new Cane("Buddy", 3, "Labrador"),
            new Gatto("Luna", 2, "Nero"),
            new Pesce("Nemo", 1)
        };
        
        System.out.println("--- Versi degli animali ---");
        for (Animale animale : animali) {
            animale.faiVerso();  // Comportamento polimorfico!
        }
    }
}

Astrazione e Classi Astratte

Definire contratti senza implementazione
L'astrazione permette di definire metodi senza implementazione (metodi astratti) che devono essere implementati dalle sottoclassi. Le classi astratte non possono essere istanziate direttamente. 
// Classe astratta - non può essere istanziata
abstract class Dipendente {
    protected String nome;
    protected String cognome;
    protected String matricola;
    
    public Dipendente(String nome, String cognome, String matricola) {
        this.nome = nome;
        this.cognome = cognome;
        this.matricola = matricola;
    }
    
    // Metodo concreto (implementato)
    public void stampaInfoBase() {
        System.out.println("Dipendente: " + nome + " " + cognome);
        System.out.println("Matricola: " + matricola);
    }
    
    // Metodo astratto - deve essere implementato dalle sottoclassi
    public abstract double calcolaStipendio();
    
    // Metodo astratto
    public abstract String getRuolo();
}

// Sottoclasse concreta - Sviluppatore
class Sviluppatore extends Dipendente {
    private double stipendioBase;
    private int oreStraordinario;
    private final double TARIFFA_STRAORDINARIO = 25.0;
    
    public Sviluppatore(String nome, String cognome, String matricola, double stipendioBase) {
        super(nome, cognome, matricola);
        this.stipendioBase = stipendioBase;
    }
    
    public void setOreStraordinario(int ore) {
        this.oreStraordinario = ore;
    }
    
    @Override
    public double calcolaStipendio() {
        return stipendioBase + (oreStraordinario * TARIFFA_STRAORDINARIO);
    }
    
    @Override
    public String getRuolo() {
        return "Sviluppatore Software";
    }
}

// Sottoclasse concreta - Manager
class Manager extends Dipendente {
    private double stipendioBase;
    private double bonus;
    private double percentualeObiettivi;
    
    public Manager(String nome, String cognome, String matricola, double stipendioBase) {
        super(nome, cognome, matricola);
        this.stipendioBase = stipendioBase;
    }
    
    public void setBonus(double bonus) {
        this.bonus = bonus;
    }
    
    public void setPercentualeObiettivi(double percentuale) {
        this.percentualeObiettivi = percentuale;
    }
    
    @Override
    public double calcolaStipendio() {
        double bonusObiettivi = stipendioBase * (percentualeObiettivi / 100);
        return stipendioBase + bonus + bonusObiettivi;
    }
    
    @Override
    public String getRuolo() {
        return "Manager";
    }
}

// Sottoclasse concreta - Venditore
class Venditore extends Dipendente {
    private double stipendioBase;
    private double provvigioni;
    
    public Venditore(String nome, String cognome, String matricola, double stipendioBase) {
        super(nome, cognome, matricola);
        this.stipendioBase = stipendioBase;
    }
    
    public void setProvvigioni(double provvigioni) {
        this.provvigioni = provvigioni;
    }
    
    @Override
    public double calcolaStipendio() {
        return stipendioBase + provvigioni;
    }
    
    @Override
    public String getRuolo() {
        return "Venditore";
    }
}

public class AstrazioneDemo {
    public static void main(String[] args) {
        Sviluppatore dev = new Sviluppatore("Mario", "Rossi", "DEV001", 2500);
        dev.setOreStraordinario(10);
        
        Manager mgr = new Manager("Laura", "Bianchi", "MGR001", 4000);
        mgr.setBonus(1500);
        mgr.setPercentualeObiettivi(10);
        
        Venditore vend = new Venditore("Giuseppe", "Verdi", "VND001", 1200);
        vend.setProvvigioni(3500);
        
        // Polimorfismo con classe astratta
        Dipendente[] dipendenti = {dev, mgr, vend};
        
        System.out.println("=== LISTA DIPENDENTI ===\n");
        for (Dipendente d : dipendenti) {
            d.stampaInfoBase();
            System.out.println("Ruolo: " + d.getRuolo());
            System.out.printf("Stipendio: €%.2f\n", d.calcolaStipendio());
            System.out.println("---");
        }
    }
}

Classe astratta vs Interfaccia:
- Classe astratta: può avere stato (attributi) e metodi concreti
- Interfaccia: solo metodi astratti (fino a Java 7), default/static da Java 8
- Una classe può estendere UNA sola classe astratta
- Una classe può implementare PIÙ interfacce

Interfacce (Interfaces)

Contratti che definiscono "cosa fare" non "come fare"
Un'interfaccia definisce un contratto che le classi devono rispettare. A differenza delle classi astratte, una classe può implementare più interfacce (ereditarietà multipla). 
// Definizione di interfacce
interface Volante {
    void vola();
    
    // Metodi default (Java 8+)
    default void plana() {
        System.out.println("Sta planando...");
    }
    
    // Metodi statici (Java 8+)
    static void decolla() {
        System.out.println("Decollo in corso...");
    }
}

interface Nuotante {
    void nuota();
}

interface Predatore {
    void caccia();
    
    default void avvistaPreda() {
        System.out.println("Preda avvistata!");
    }
}

interface Riproducibile {
    void riproduci();
}

// Classe che implementa più interfacce
class Aquila implements Volante, Predatore, Riproducibile {
    private String nome;
    
    public Aquila(String nome) {
        this.nome = nome;
    }
    
    @Override
    public void vola() {
        System.out.println(nome + " vola maestosamente nel cielo!");
    }
    
    @Override
    public void caccia() {
        System.out.println(nome + " scende in picchiata per catturare la preda!");
    }
    
    @Override
    public void riproduci() {
        System.out.println(nome + " costruisce un nido e depone le uova.");
    }
}

class Pinguino implements Volante, Nuotante {
    private String nome;
    
    public Pinguino(String nome) {
        this.nome = nome;
    }
    
    @Override
    public void vola() {
        System.out.println(nome + " non può volare, ma fa delle belle nuotate!");
    }
    
    @Override
    public void nuota() {
        System.out.println(nome + " nuota rapidamente nell'acqua!");
    }
}

class Squalo implements Nuotante, Predatore {
    private String nome;
    
    public Squalo(String nome) {
        this.nome = nome;
    }
    
    @Override
    public void nuota() {
        System.out.println(nome + " nuota minacciosamente!");
    }
    
    @Override
    public void caccia() {
        System.out.println(nome + " cerca la sua preda!");
    }
}

// Interfaccia funzionale (un solo metodo astratto) - per lambda
@FunctionalInterface
interface OperazioneMatematica {
    int esegui(int a, int b);
}

// Interfaccia con ereditarietà multipla tra interfacce
interface AnimaleMarino extends Nuotante, Riproducibile {
    void respiraSottAcqua();
}

class Delfino implements AnimaleMarino {
    private String nome;
    
    public Delfino(String nome) {
        this.nome = nome;
    }
    
    @Override
    public void nuota() {
        System.out.println(nome + " nuota elegantemente!");
    }
    
    @Override
    public void riproduci() {
        System.out.println(nome + " partorisce un cucciolo.");
    }
    
    @Override
    public void respiraSottAcqua() {
        System.out.println(nome + " risale in superficie per respirare.");
    }
}

public class InterfacceDemo {
    public static void main(String[] args) {
        System.out.println("=== ESEMPI INTERFACCE ===\n");
        
        Aquila aquila = new Aquila("Aquila reale");
        Pinguino pinguino = new Pinguino("Pinguino imperatore");
        Squalo squalo = new Squalo("Grande squalo bianco");
        Delfino delfino = new Delfino("Delfino");
        
        aquila.vola();
        aquila.caccia();
        aquila.plana();      // Metodo default
        aquila.riproduci();
        Volante.decolla();    // Metodo statico
        
        System.out.println();
        pinguino.vola();
        pinguino.nuota();
        
        System.out.println();
        squalo.nuota();
        squalo.caccia();
        
        System.out.println();
        delfino.nuota();
        delfino.riproduci();
        delfino.respiraSottAcqua();
        
        // ========== INTERFACCIA FUNZIONALE CON LAMBDA ==========
        System.out.println("\n=== Interfacce funzionali e Lambda ===");
        
        OperazioneMatematica somma = (a, b) -> a + b;
        OperazioneMatematica moltiplica = (a, b) -> a * b;
        OperazioneMatematica massimo = (a, b) -> a > b ? a : b;
        
        System.out.println("Somma: " + somma.esegui(10, 5));
        System.out.println("Moltiplicazione: " + moltiplica.esegui(10, 5));
        System.out.println("Massimo: " + massimo.esegui(10, 5));
    }
}

public class Studente {
    private String nome;
    private String cognome;
    private int matricola;
    private double mediaVoti;
    private static int contatoreMatricole = 1000;
    
    // Blocco di inizializzazione (eseguito prima del costruttore)
    {
        System.out.println("Blocco di inizializzazione eseguito");
        this.matricola = ++contatoreMatricole;
    }
    
    // Blocco statico (eseguito una sola volta quando la classe viene caricata)
    static {
        System.out.println("Blocco statico - caricamento classe Studente");
        contatoreMatricole = 1000;
    }
    
    // Costruttore senza parametri (default)
    public Studente() {
        this("Sconosciuto", "Sconosciuto");
        System.out.println("Costruttore default chiamato");
    }
    
    // Costruttore con parametri
    public Studente(String nome, String cognome) {
        this(nome, cognome, 0.0);
        System.out.println("Costruttore con 2 parametri chiamato");
    }
    
    // Costruttore completo
    public Studente(String nome, String cognome, double mediaVoti) {
        this.nome = nome;
        this.cognome = cognome;
        this.mediaVoti = mediaVoti;
        System.out.println("Costruttore completo chiamato");
    }
    
    // Costruttore di copia
    public Studente(Studente altro) {
        this(altro.nome, altro.cognome, altro.mediaVoti);
        System.out.println("Costruttore di copia chiamato");
    }
    
    // Costruttore privato (pattern Singleton)
    private Studente(int matricolaSpeciale) {
        this.matricola = matricolaSpeciale;
        this.nome = "Speciale";
        this.cognome = "Speciale";
    }
    
    public static Studente creaStudenteSpeciale() {
        return new Studente(9999);
    }
    
    public void stampaInfo() {
        System.out.println("Studente: " + nome + " " + cognome);
        System.out.println("Matricola: " + matricola);
        System.out.println("Media voti: " + mediaVoti);
    }
    
    public static void main(String[] args) {
        System.out.println("--- Creazione oggetti ---\n");
        
        Studente s1 = new Studente();
        System.out.println();
        
        Studente s2 = new Studente("Mario", "Rossi");
        System.out.println();
        
        Studente s3 = new Studente("Laura", "Bianchi", 28.5);
        System.out.println();
        
        Studente s4 = new Studente(s3);  // Costruttore di copia
        System.out.println();
        
        Studente s5 = Studente.creaStudenteSpeciale();
        
        System.out.println("\n--- Informazioni studenti ---\n");
        s1.stampaInfo();
        System.out.println();
        s2.stampaInfo();
        System.out.println();
        s3.stampaInfo();
        System.out.println();
        s4.stampaInfo();
        System.out.println();
        s5.stampaInfo();
    }
}

Membri Statici

Appartengono alla classe, non alle istanze
public class Contatore {
    // Variabile di istanza (ogni oggetto ha la sua copia)
    private int contatoreIstanza;
    
    // Variabile statica (una sola copia condivisa tra tutti gli oggetti)
    private static int contatoreGlobale = 0;
    
    // Costante statica (public static final)
    public static final String NOME_CLASSE = "Contatore";
    public static final int VALORE_MASSIMO = 100;
    
    // Costruttore
    public Contatore() {
        contatoreIstanza = 0;
        contatoreGlobale++;
        System.out.println("Creato oggetto #" + contatoreGlobale);
    }
    
    // Metodo di istanza
    public void incrementaIstanza() {
        if (contatoreIstanza < VALORE_MASSIMO) {
            contatoreIstanza++;
        }
    }
    
    public int getContatoreIstanza() {
        return contatoreIstanza;
    }
    
    // Metodo statico (può accedere solo a membri statici)
    public static int getContatoreGlobale() {
        return contatoreGlobale;
    }
    
    public static String getNomeClasse() {
        return NOME_CLASSE;
    }
    
    public static void resetContatoreGlobale() {
        contatoreGlobale = 0;
    }
    
    // Esempio di classe statica interna (nested class)
    public static class Utility {
        public static void stampaInfo() {
            System.out.println("Classe: " + NOME_CLASSE);
            System.out.println("Valore massimo: " + VALORE_MASSIMO);
            System.out.println("Contatori creati: " + contatoreGlobale);
        }
    }
}

public class StaticDemo {
    public static void main(String[] args) {
        // Accesso a membri statici senza creare oggetti
        System.out.println("Nome classe: " + Contatore.getNomeClasse());
        System.out.println("Valore massimo: " + Contatore.VALORE_MASSIMO);
        
        // Creazione oggetti
        Contatore c1 = new Contatore();
        Contatore c2 = new Contatore();
        Contatore c3 = new Contatore();
        
        // Operazioni sulle istanze
        c1.incrementaIstanza();
        c1.incrementaIstanza();
        c2.incrementaIstanza();
        
        System.out.println("\n--- Contatori istanza ---");
        System.out.println("c1: " + c1.getContatoreIstanza());
        System.out.println("c2: " + c2.getContatoreIstanza());
        System.out.println("c3: " + c3.getContatoreIstanza());
        
        // Accesso a membri statici
        System.out.println("\n--- Membri statici ---");
        System.out.println("Totale oggetti creati: " + Contatore.getContatoreGlobale());
        
        // Classe statica interna
        System.out.println("\n--- Classe statica interna ---");
        Contatore.Utility.stampaInfo();
        
        // Esempio pratico: contatore condiviso
        System.out.println("\n--- Contatore condiviso tra thread ---");
        Runnable task = () -> {
            for (int i = 0; i < 100; i++) {
                Contatore.getContatoreGlobale();  // solo lettura
            }
            System.out.println(Thread.currentThread().getName() + " completato");
        };
        
        Thread t1 = new Thread(task, "Thread-1");
        Thread t2 = new Thread(task, "Thread-2");
        t1.start();
        t2.start();
    }
}

Regole dei membri statici:
- I membri statici appartengono alla classe, non alle istanze
- Si accede tramite Classe.membro (non serve un oggetto)
- I metodi statici possono accedere solo a membri statici
- static final definisce costanti (convenzione: NOME_MAIUSCOLO)
- Le classi statiche interne possono essere istanziate senza un'istanza della classe esterna

import java.util.*;

// ==================== CLASSE ASTRATTA ====================
abstract class MaterialeBibliotecario {
    protected String titolo;
    protected String codice;
    protected boolean disponibile;
    protected LocalDate dataInserimento;
    
    public MaterialeBibliotecario(String titolo, String codice) {
        this.titolo = titolo;
        this.codice = codice;
        this.disponibile = true;
        this.dataInserimento = LocalDate.now();
    }
    
    public abstract String getTipo();
    public abstract int getGiorniPrestito();
    
    public void presta() {
        if (disponibile) {
            disponibile = false;
            System.out.println("📖 " + titolo + " è stato prestato");
        } else {
            System.out.println("❌ " + titolo + " non è disponibile");
        }
    }
    
    public void restituisci() {
        disponibile = true;
        System.out.println("📚 " + titolo + " è stato restituito");
    }
    
    public void stampaInfo() {
        System.out.println("[" + getTipo() + "] " + titolo + " (Codice: " + codice + ")");
        System.out.println("  Disponibile: " + (disponibile ? "✅ Sì" : "❌ No"));
        System.out.println("  Inserito il: " + dataInserimento);
        System.out.println("  Giorni prestito: " + getGiorniPrestito());
    }
}

class Biblioteca {
    private String nome;
    private String indirizzo;
    private List<MaterialeBibliotecario> materiali;
    private Map<String, Utente> utentiRegistrati;
    
    public Biblioteca(String nome, String indirizzo) {
        this.nome = nome;
        this.indirizzo = indirizzo;
        this.materiali = new ArrayList<>();
        this.utentiRegistrati = new HashMap<>();
    }
    
    public void aggiungiMateriale(MaterialeBibliotecario materiale) {
        materiali.add(materiale);
        System.out.println("✅ Aggiunto: " + materiale.titolo);
    }
    
    public void rimuoviMateriale(String codice) {
        materiali.removeIf(m -> m.codice.equals(codice));
        System.out.println("🗑️ Rimosso materiale con codice: " + codice);
    }
    
    public MaterialeBibliotecario cercaPerCodice(String codice) {
        for (MaterialeBibliotecario m : materiali) {
            if (m.codice.equals(codice)) {
                return m;
            }
        }
        return null;
    }
    
    public List<MaterialeBibliotecario> cercaPerTitolo(String titolo) {
        List<MaterialeBibliotecario> risultati = new ArrayList<>();
        for (MaterialeBibliotecario m : materiali) {
            if (m.titolo.toLowerCase().contains(titolo.toLowerCase())) {
                risultati.add(m);
            }
        }
        return risultati;
    }
    
    public List<MaterialeBibliotecario> getMaterialiDisponibili() {
        List<MaterialeBibliotecario> disponibili = new ArrayList<>();
        for (MaterialeBibliotecario m : materiali) {
            if (m.disponibile) {
                disponibili.add(m);
            }
        }
        return disponibili;
    }
    
    public void stampaCatalogo() {
        System.out.println("\n📚 CATALOGO DELLA BIBLIOTECA '" + nome + "' 📚");
        System.out.println("Indirizzo: " + indirizzo);
        System.out.println("Totale materiali: " + materiali.size());
        System.out.println("Disponibili: " + getMaterialiDisponibili().size());
        System.out.println("\n--- Elenco materiali ---");
        
        for (MaterialeBibliotecario m : materiali) {
            m.stampaInfo();
            System.out.println();
        }
    }
    
    public void registraUtente(Utente utente) {
        utentiRegistrati.put(utente.getCodiceFiscale(), utente);
        System.out.println("✅ Utente registrato: " + utente.getNome() + " " + utente.getCognome());
    }
}

// ==================== CLASSE UTENTE ====================
class Utente {
    private String nome;
    private String cognome;
    private String codiceFiscale;
    private String email;
    private List<MaterialeBibliotecario> prestitiAttivi;
    
    public Utente(String nome, String cognome, String codiceFiscale, String email) {
        this.nome = nome;
        this.cognome = cognome;
        this.codiceFiscale = codiceFiscale;
        this.email = email;
        this.prestitiAttivi = new ArrayList<>();
    }
    
    public void prendiInPrestito(MaterialeBibliotecario materiale) {
        if (materiale.disponibile) {
            materiale.presta();
            prestitiAttivi.add(materiale);
        } else {
            System.out.println("❌ " + materiale.titolo + " non è disponibile per il prestito");
        }
    }
    
    public void restituisci(MaterialeBibliotecario materiale) {
        if (prestitiAttivi.remove(materiale)) {
            materiale.restituisci();
        } else {
            System.out.println("❌ " + materiale.titolo + " non è in prestito da questo utente");
        }
    }
    
    public void stampaPrestiti() {
        System.out.println("\n📋 Prestiti attivi di " + nome + " " + cognome + ":");
        if (prestitiAttivi.isEmpty()) {
            System.out.println("  Nessun prestito attivo");
        } else {
            for (MaterialeBibliotecario m : prestitiAttivi) {
                System.out.println("  - " + m.titolo + " (scadenza: " + 
                    LocalDate.now().plusDays(m.getGiorniPrestito()) + ")");
            }
        }
    }
    
    // Getter
    public String getNome() { return nome; }
    public String getCognome() { return cognome; }
    public String getCodiceFiscale() { return codiceFiscale; }
    public String getEmail() { return email; }
}
