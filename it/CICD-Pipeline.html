    Glossario CI/CD

    Continuous Integration (CI)

    Pratica di integrare frequentemente il codice in un repository condiviso, con build e test automatici.

    Continuous Delivery (CD)

    Estensione della CI: il software è sempre pronto per il deployment in produzione.

    Continuous Deployment

    Ogni modifica che supera i test viene automaticamente deployata in produzione.

    Pipeline

    Sequenza automatizzata di step che il codice attraversa dal commit al deploy.

    Artifact

    Output della build (JAR, Docker image, bundle) pronto per il deployment.

    Orchestrator

    Strumento che gestisce l'esecuzione della pipeline (Jenkins, GitLab CI, GitHub Actions).
